# Labs Directory

Workspace for external repository analysis (GitHub Scout discoveries)

## Structure

```
labs/
  ├── README.md (this file)
  ├── repos/      (cloned repositories)
  └── reports/    (analysis reports)
```

## Usage

This directory is used by Consilium system to:
1. Clone discovered repositories from GitHub Scout
2. Analyze code structure and philosophy
3. Generate reports for agent review (Arianna, Monday, Claude Defender)

Repos here are temporary - for autopsy, not integration.
